### ida 快捷键
- 空格键 在hex-ray中进行列表视图与图形视图的转换
- TAB 在hex-ray中进行汇编指令和伪代码转换
- ESC 翻页，返回前一页面
- G 定位地址
- X 交叉引用
- N 重命名
- Y 修改函数原型或变量类型
- / 注释
- * 把变量重定义为数组
- shift + f12 搜索字符串
- D 双击进变量之后，可以对变量的数据类型进行切换/在结构体ends处新建成员
- R 字符转换，如'95'->'_'
- H 将字符在十进制和十六进制之间转换
- P 创建函数
- shift + f9 进入结构体界面
- insert 在结构体界面创建结构体
- alt + q 对指定数据应用结构体


### C++ OOP 对象在IDA中的表现形式

对象是类的一个实例，一个对象需要存储它的内存，所以可以在IDA中看调用new()或者栈分配内存的部分。

以2024长城杯初赛的题anote为例，在add一个新note时，其反汇编c++代码有如下表现形式：

```cpp
v1 = (_DWORD *)operator new(28u);
```

在申请完内存后，程序马上会调用构造函数对这块内存进行初始化，其表现形式为：

```cpp
v1 = (_DWORD *)operator new(28u);
for ( i = 0; i < 7; ++i )
v1[i] = 0;
sub_8048DFC(v1);
```

可以看到，`sub_8048DFC`应当就是该对象创建时会调用的构造函数

### 逆向数组

形如`*(p + index*offset)`形式大概率是数组

处理：双击进变量之后按下`*`进行数组创建

跳出来的窗口有几个字段值得注意：

- Array size：数组中元素总数
- Items on a line：一行中的元素数；0表示适量
- Element print width：每个元素打印的宽度
